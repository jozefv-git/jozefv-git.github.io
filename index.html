    Anonym Chat Application - STUDENT PROJECT

    Mobile chat application which preserves users’ privacy by their wish. Anyone who is not logged in can view, open, chat and interact in public chatrooms. Users can also create account via registration screen. With registered accounts they can create chatrooms (public / private). Registered / Private users can interact between each other (answer desired questions and like it). Registered user can also see profile of another registered users or create friendships and have personal chat with them. Registered users have possibility to edit their profile – change username to act like private – in this case, selected user cannot be added by anyone to friend list and he / she is acting like anonym user. User profile contains statistics about number of created posts, likes received and achievements based on these values.

    Voting Questionnaire - STUDENT PROJECT

    Fully responsive web page with registration and different levels of authentication - Administrator, Politician, Non - logged user. For availability to register as a politician you need to have invitation code from administrator. As an administrator you have privileges to add / delete new politician and also possibility to update questions provided in the questionnaire. As an administrator you have also possibility to see detailed statistic page with data gained from anonymous (non - logged) users. As a politician you have your own profile + possibility to answer the questions. Answers to questions can be also updated later. If politician answered all questions, he or she can be shown as one of three possible voting candidate for non - logged user (if the answers for the questions are matching). Best matching candidate will be shown to user. User can also see result page with detailed comparison between his or her answers and politician answers.
