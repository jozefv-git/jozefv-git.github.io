    Anonym Chat Application - STUDENT PROJECT

    Mobile chat application which preserves users’ privacy by their wish. Anyone who is not logged in can view, open, chat and interact in public chatrooms. Users can also create account via registration screen. With registered accounts they can create chatrooms (public / private). Registered / Private users can interact between each other (answer desired questions and like it). Registered user can also see profile of another registered users or create friendships and have personal chat with them. Registered users have possibility to edit their profile – change username to act like private – in this case, selected user cannot be added by anyone to friend list and he / she is acting like anonym user. User profile contains statistics about number of created posts, likes received and achievements based on these values.
